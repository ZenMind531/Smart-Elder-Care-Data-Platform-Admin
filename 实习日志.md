## 第1天

**日期：** 2026年7月7日

**今日工作内容：**

1. **学习 MyBatis 框架基础概念**
   - 了解了 SSM（Spring + SpringMVC + MyBatis）和 SSH（Spring + Struts + Hibernate）两种主流 Java 技术栈
   - 学习了 ORM（对象关系映射）的核心思想：一个数据表对应一个 Java 实体类，数据表的字段对应实体类的属性
   - 理解了持久化的概念：程序数据在瞬时状态（内存）和持久状态（数据库/文件）之间的转换过程

2. **搭建 MyBatis 开发环境**
   - 使用 Maven 创建项目，配置 pom.xml 引入 MyBatis、MySQL 驱动、Log4j 日志、JUnit 测试等依赖
   - 创建数据库实体类（POJO）：Student.java，字段与数据库表列一一对应
   - 创建 DAO 层接口（StudentDao）和 SQL 映射文件（StudentMapper.xml）
   - 配置 MyBatis 核心控制文件（mybatis-config.xml），包含数据源、驼峰命名规则、类型别名、JDBC 连接、Mapper 加载
   - 配置 jdbc.properties 数据库连接信息和 log4j.properties 日志输出

3. **编写测试类验证环境**
   - 创建 SqlSessionFactory（静态代码块，全局唯一）
   - 通过 SqlSession 执行数据库操作
   - 学习了两种调用方式：传统 selectOne 方式和 DAO 映射 getMapper 方式
   - 成功从数据库查询学生信息并打印到控制台

**遇到问题及解决：**
- pom.xml 中 `</project>` 标签位置错误导致依赖配置失效，将 `<dependencies>` 放入 `<project>` 标签内部解决
- MySQL 驱动版本过旧（5.1.29）导致连接新版 MySQL 失败，升级到 8.0.33 并修改驱动类名为 `com.mysql.cj.jdbc.Driver` 解决
