## 〇、目标与原则

### 安全三原则
1. **最小权限**：用户只能干他该干的事。
2. **纵深防御**：一道防线被破，后面还有。
3. **失败安全**：出问题默认拒绝，不要默认放行。

##### 2) 工具类 `BCryptUtil.java`

```java
package com.smarteldercare.common.security;

import org.springframework.security.crypto.bcrypt.BCrypt;

/**
 * 密码哈希工具（BCrypt，自带盐）。
 * 同一个密码每次哈希结果都不同，但能正确验证。
 */
public final class BCryptUtil {
    private BCryptUtil() {}
    private static final int COST = 12; // 2^12 = 4096 次迭代，越大越慢越安全

    public static String hash(String rawPassword) {
        if (rawPassword == null || rawPassword.isEmpty()) {
            throw new IllegalArgumentException("密码不能为空");
        }
        return BCrypt.hashpw(rawPassword, BCrypt.gensalt(COST));
    }

    public static boolean matches(String rawPassword, String hashedPassword) {
        if (rawPassword == null || hashedPassword == null) return false;
        try {
            return BCrypt.checkpw(rawPassword, hashedPassword);
        } catch (Exception e) {
            return false;
        }
    }
}
```

##### 3) 改登录 `AuthServiceImpl.login()`
```java

    if (user == null) {

    if (!BCryptUtil.matches(password, user.getPassword())) {

    }
    // ...生成 token
}
```


##### 5) 改密码 `AuthServiceImpl.changePassword()`
```java
public void changePassword(Long userId, String oldPwd, String newPwd) {
    SysUser user = userMapper.selectById(userId);
    if (!BCryptUtil.matches(oldPwd, user.getPassword())) {
        throw new BusinessException(400, "原密码错误");
    }
    user.setPassword(BCryptUtil.hash(newPwd));
    userMapper.updateById(user);

```

#### 数据迁移（重要！）

老库里的明文/旧哈希密码**不能直接用**，需要强制重置：

```sql
-- 1. 把所有密码临时置为 NULL
UPDATE sys_user SET password = NULL;

-- 2. 业务上：登录时检测到 password 为 NULL，跳转"首次登录设置密码"页面
--    或者：管理员后台批量重置，发邮件让用户重新设置
```

> 不要写"自动用明文密码登录然后再哈希"的代码——这是把明文密码在内存里跑一遍，留痕。

#### 验证

```bash

# 2. 用错误密码登录，返回 401
# 3. 用正确密码登录，返回 token
# 4. 修改密码后，旧密码失效
```

### 1.2 SQL 注入防护

#### 问题
任何用 `${}` 拼接 SQL 的地方都有 SQL 注入风险。

#### 修复

**MyBatis-Plus 默认 `#{}` 是预编译，安全的。** 排查所有 mapper：

```java
// ❌ 错误：${} 直接拼接
@Select("SELECT * FROM user WHERE name = '${name}'")
User findByName(String name);

// ✅ 正确：#{} 预编译
@Select("SELECT * FROM user WHERE name = #{name}")
User findByName(String name);
```

**LambdaQueryWrapper 不要用 `last()` 拼字符串**：
```java
// ❌ 危险
wrapper.last("LIMIT " + limit);
// ✅ 安全
wrapper.last("LIMIT " + Integer.parseInt(limit)); // 强转成 int
```

**模糊查询关键字过滤**：
```java
// 关键词里有 % 或 _ 时需要转义
String safe = keyword.replace("%", "\\%").replace("_", "\\_");
wrapper.like("name", safe);
```

#### 验证
```bash
# 用 SQL 注入测试
curl "http://localhost:8080/api/elderly?keyword=' OR 1=1 --"
# 预期：返回空或 400，不能返回全表
```

### 1.3 Spring Security + JWT 鉴权

#### 问题
当前接口任何人都能调。需要**登录后用 token 访问**。

#### 修复

##### 1) 配置 `application.yml`
```yaml

  # 32 字节以上随机字符串，生产环境从环境变量读
  secret: ${JWT_SECRET:please-change-me-to-a-random-32-byte-string!!!}
  # token 有效期
  expire-minutes: 120
  # 请求头名称
  header: Authorization
  # token 前缀

```

##### 2) JWT 工具 `JwtUtil.java`

```java
package com.smarteldercare.common.security;

import io.jsonwebtoken.Claims;
import io.jsonwebtoken.Jwts;
import io.jsonwebtoken.security.Keys;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Component;

import javax.crypto.SecretKey;
import java.nio.charset.StandardCharsets;
import java.util.Date;
import java.util.HashMap;
import java.util.Map;

@Component
public class JwtUtil {

    @Value("${jwt.secret}")
    private String secret;

    @Value("${jwt.expire-minutes}")
    private long expireMinutes;

    private SecretKey getKey() {
        return Keys.hmacShaKeyFor(secret.getBytes(StandardCharsets.UTF_8));
    }

    /** 生成 token */
    public String generate(Long userId, String username, String role) {
        Map<String, Object> claims = new HashMap<>();
        claims.put("uid", userId);
        claims.put("username", username);
        claims.put("role", role);

        long now = System.currentTimeMillis();
        return Jwts.builder()
                .claims(claims)
                .subject(username)
                .issuedAt(new Date(now))
                .expiration(new Date(now + expireMinutes * 60 * 1000))
                .signWith(getKey())
                .compact();
    }

    /** 解析 token，失败抛异常 */
    public Claims parse(String token) {
        return Jwts.parser()
                .verifyWith(getKey())
                .build()
                .parseSignedClaims(token)
                .getPayload();
    }

    public long getExpireSeconds() {
        return expireMinutes * 60;
    }
}
```

##### 3) JWT 过滤器 `JwtAuthenticationFilter.java`

```java
package com.smarteldercare.common.security;

import io.jsonwebtoken.Claims;
import jakarta.servlet.FilterChain;
import jakarta.servlet.ServletException;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import org.springframework.security.authentication.UsernamePasswordAuthenticationToken;
import org.springframework.security.core.authority.SimpleGrantedAuthority;
import org.springframework.security.core.context.SecurityContextHolder;
import org.springframework.security.web.authentication.WebAuthenticationDetailsSource;
import org.springframework.stereotype.Component;
import org.springframework.web.filter.OncePerRequestFilter;

import java.io.IOException;
import java.util.List;

@Component
public class JwtAuthenticationFilter extends OncePerRequestFilter {

    private final JwtUtil jwtUtil;

    public JwtAuthenticationFilter(JwtUtil jwtUtil) {
        this.jwtUtil = jwtUtil;
    }

    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain chain) throws ServletException, IOException {
        String header = request.getHeader("Authorization");
        if (header != null && header.startsWith("Bearer ")) {
            String token = header.substring(7);
            try {
                Claims claims = jwtUtil.parse(token);
                String username = claims.getSubject();
                String role = (String) claims.get("role");

                if (username != null && SecurityContextHolder.getContext().getAuthentication() == null) {
                    UsernamePasswordAuthenticationToken auth =
                            new UsernamePasswordAuthenticationToken(
                                    username,
                                    null,
                                    List.of(new SimpleGrantedAuthority("ROLE_" + role))
                            );
                    auth.setDetails(new WebAuthenticationDetailsSource().buildDetails(request));
                    SecurityContextHolder.getContext().setAuthentication(auth);
                }
            } catch (Exception e) {
                // token 过期/伪造/篡改，不抛，让 SecurityConfig 处理为 401
                SecurityContextHolder.clearContext();
            }
        }
        chain.doFilter(request, response);
    }
}
```

##### 4) Security 配置 `SecurityConfig.java`

```java
package com.smarteldercare.common.security;

import com.fasterxml.jackson.databind.ObjectMapper;
import jakarta.servlet.http.HttpServletResponse;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.http.MediaType;
import org.springframework.security.config.annotation.method.configuration.EnableMethodSecurity;
import org.springframework.security.config.annotation.web.builders.HttpSecurity;
import org.springframework.security.config.annotation.web.configurers.AbstractHttpConfigurer;
import org.springframework.security.config.http.SessionCreationPolicy;
import org.springframework.security.web.SecurityFilterChain;
import org.springframework.security.web.authentication.UsernamePasswordAuthenticationFilter;

import java.util.Map;

@Configuration
@EnableMethodSecurity // 开启 @PreAuthorize
public class SecurityConfig {

    private final JwtAuthenticationFilter jwtFilter;
    private final CorsConfig corsConfig;
    private final ObjectMapper objectMapper = new ObjectMapper();

    public SecurityConfig(JwtAuthenticationFilter jwtFilter, CorsConfig corsConfig) {
        this.jwtFilter = jwtFilter;
        this.corsConfig = corsConfig;
    }

    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .csrf(AbstractHttpConfigurer::disable)
            .cors(c -> c.configurationSource(corsConfig.corsConfigurationSource()))
            .sessionManagement(s -> s.sessionCreationPolicy(SessionCreationPolicy.STATELESS))
            .authorizeHttpRequests(auth -> auth
                // 完全公开的接口
                .requestMatchers(
                    "/api/auth/login",
                    "/api/auth/register",
                    "/api/health",            // 健康检查
                    "/error"
                ).permitAll()
                // 静态资源（如有）
                .requestMatchers("/uploads/**", "/static/**").permitAll()
                // 其他全部要登录
                .anyRequest().authenticated()
            )
            .exceptionHandling(ex -> ex
                .authenticationEntryPoint((req, res, e) -> writeError(res, 401, "未登录或 token 失效"))
                .accessDeniedHandler((req, res, e) -> writeError(res, 403, "无权限"))
            )
            .addFilterBefore(jwtFilter, UsernamePasswordAuthenticationFilter.class);

        return http.build();
    }

    private void writeError(HttpServletResponse res, int code, String msg) throws java.io.IOException {
        res.setStatus(code);
        res.setContentType(MediaType.APPLICATION_JSON_VALUE);
        res.setCharacterEncoding("UTF-8");
        res.getWriter().write(objectMapper.writeValueAsString(
                Map.of("code", code, "message", msg, "timestamp", System.currentTimeMillis())
        ));
    }
}
```

```java
package com.smarteldercare.common.security.exception;

import com.smarteldercare.common.result.ApiResponse;
import jakarta.servlet.http.HttpServletRequest;
import lombok.extern.slf4j.Slf4j;
import org.springframework.http.converter.HttpMessageNotReadableException;
import org.springframework.validation.BindException;
import org.springframework.validation.FieldError;
import org.springframework.web.bind.MethodArgumentNotValidException;
import org.springframework.web.bind.annotation.ExceptionHandler;
import org.springframework.web.bind.annotation.RestControllerAdvice;

import java.util.stream.Collectors;

@Slf4j
@RestControllerAdvice
public class GlobalExceptionHandler {

    @ExceptionHandler(BusinessException.class)
    public ApiResponse<Void> handleBusiness(BusinessException e, HttpServletRequest req) {
        log.warn("业务异常 [{} {}] code={} msg={}",
                req.getMethod(), req.getRequestURI(), e.getCode(), e.getMessage());
        return ApiResponse.fail(e.getCode(), e.getMessage());
    }

    @ExceptionHandler(MethodArgumentNotValidException.class)
    public ApiResponse<Void> handleValid(MethodArgumentNotValidException e) {
        String msg = e.getBindingResult().getFieldErrors().stream()
                .map(FieldError::getDefaultMessage)
                .collect(Collectors.joining("; "));
        return ApiResponse.fail(400, msg);
    }

    @ExceptionHandler(BindException.class)
    public ApiResponse<Void> handleBind(BindException e) {
        String msg = e.getBindingResult().getFieldErrors().stream()
                .map(FieldError::getDefaultMessage)
                .collect(Collectors.joining("; "));
        return ApiResponse.fail(400, msg);
    }

    @ExceptionHandler(HttpMessageNotReadableException.class)
    public ApiResponse<Void> handleBadJson(HttpMessageNotReadableException e) {
        return ApiResponse.fail(400, "请求体格式错误");
    }

    /** 兜底：所有未捕获异常，不暴露细节 */
    @ExceptionHandler(Exception.class)
    public ApiResponse<Void> handleAny(Exception e, HttpServletRequest req) {
        // 详细堆栈写日志
        log.error("系统异常 [{} {}]", req.getMethod(), req.getRequestURI(), e);
        // 对外只说"系统繁忙"
        return ApiResponse.fail(500, "系统繁忙，请稍后重试");
    }
}
```

### 2.1 CORS 白名单

**绝对别用 `*` 配生产环境**。改成只允许你的前端域名：

##### `CorsConfig.java`

```java
package com.smarteldercare.common.security;

import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.web.cors.CorsConfiguration;
import org.springframework.web.cors.CorsConfigurationSource;
import org.springframework.web.cors.UrlBasedCorsConfigurationSource;

import java.util.List;

@Configuration
public class CorsConfig {

    @Bean
    public CorsConfigurationSource corsConfigurationSource() {
        CorsConfiguration cfg = new CorsConfiguration();
        // 改成你的真实前端域名
        cfg.setAllowedOrigins(List.of(
                "https://smart-eldercare.online",
                "https://admin.smart-eldercare.online"
        ));
        cfg.setAllowedMethods(List.of("GET","POST","PUT","DELETE","PATCH","OPTIONS"));
        cfg.setAllowedHeaders(List.of("*"));
        cfg.setExposedHeaders(List.of("Authorization"));
        cfg.setAllowCredentials(true);
        cfg.setMaxAge(3600L);

        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/api/**", cfg);
        return source;
    }
```

### 2.2 登录限流（防爆破）

**目标**：同一 IP 5 分钟内密码错 5 次 → 锁 10 分钟。

##### 1) 简易实现（基于内存，适合单机）

```java
package com.smarteldercare.common.security;

import java.util.concurrent.ConcurrentHashMap;
import java.util.concurrent.atomic.AtomicInteger;

/**
 * 滑动窗口限流器（内存版）。
 * 多实例部署请改用 Redis。
 */
public class RateLimiter {
    private static final int MAX_ATTEMPTS = 5;
    private static final long WINDOW_MS = 5 * 60 * 1000;  // 5 分钟
    private static final long LOCK_MS = 10 * 60 * 1000;   // 锁定 10 分钟

    private static final ConcurrentHashMap<String, Attempt> MAP = new ConcurrentHashMap<>();

    public static class TooManyException extends RuntimeException {
        public TooManyException(String msg) { super(msg); }
    }

    private static class Attempt {
        AtomicInteger count = new AtomicInteger(0);
        long firstAt = System.currentTimeMillis();
        long lockUntil = 0;
    }

    public static void check(String key) {
        Attempt a = MAP.computeIfAbsent(key, k -> new Attempt());
        long now = System.currentTimeMillis();

        if (a.lockUntil > now) {
            long left = (a.lockUntil - now) / 1000;
            throw new TooManyException("尝试次数过多，请 " + left + " 秒后再试");
        }
        if (now - a.firstAt > WINDOW_MS) {
            a.count.set(0);
            a.firstAt = now;
        }
        if (a.count.incrementAndGet() > MAX_ATTEMPTS) {
            a.lockUntil = now + LOCK_MS;
            throw new TooManyException("尝试次数过多，已锁定 10 分钟");
        }
    }

    /** 登录成功后清零 */
    public static void clear(String key) {
        MAP.remove(key);
    }
}
```

##### 2) 在 `AuthService.login()` 加
```java
String key = "login:" + username; // 或 "login:ip:" + clientIp
try {
    RateLimiter.check(key);
} catch (RateLimiter.TooManyException e) {
    throw BusinessException.forbidden(e.getMessage());
}

if (!BCryptUtil.matches(password, user.getPassword())) {
    // 错误时 RateLimiter 已 +1，不用再手动加
    throw BusinessException.unauthorized("用户名或密码错误");
}

RateLimiter.clear(key); // 成功清零
```

### 2.3 接口防重放（timestamp + nonce）

**目标**：防止 token 被偷走后被反复使用。

##### 实现思路
1. 前端每次请求带 `timestamp`（毫秒）和 `nonce`（随机字符串）。
2. 后端校验：时间戳在 5 分钟内 + nonce 在 Redis 没出现过 → 放行。
3. nonce 在 Redis 存 5 分钟自动过期。

##### `ReplayAttackFilter.java`
```java
package com.smarteldercare.common.security;

import jakarta.servlet.*;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletRequestWrapper;
import org.springframework.data.redis.core.StringRedisTemplate;
import org.springframework.stereotype.Component;

import java.io.IOException;
import java.util.Collections;
import java.util.Enumeration;
import java.util.HashMap;
import java.util.Map;

/**
 * 简单实现：把 timestamp 和 nonce 放在 Header 里。
 * 真实项目用 Redis 存 nonce 防重放。
 */
@Component
public class ReplayAttackFilter implements Filter {
    private final StringRedisTemplate redis;
    private static final long SKEW_MS = 5 * 60 * 1000; // 5 分钟

    public ReplayAttackFilter(StringRedisTemplate redis) { this.redis = redis; }

    @Override
    public void doFilter(ServletRequest request, ServletResponse response, FilterChain chain)
            throws IOException, ServletException {
        HttpServletRequest req = (HttpServletRequest) request;
        String ts = req.getHeader("X-Timestamp");
        String nonce = req.getHeader("X-Nonce");

        // 登录等公开接口跳过
        String path = req.getRequestURI();
        if (path.contains("/auth/login") || path.contains("/auth/register")) {
            chain.doFilter(request, response);
            return;
        }

        if (ts == null || nonce == null) {
            chain.doFilter(request, response);
            return; // 没传就放行，靠 JWT 兜底
        }

        try {
            long t = Long.parseLong(ts);
            if (Math.abs(System.currentTimeMillis() - t) > SKEW_MS) {
                ((jakarta.servlet.http.HttpServletResponse) response)
                    .sendError(401, "请求已过期");
                return;
            }
            String key = "nonce:" + nonce;
            Boolean first = redis.opsForValue().setIfAbsent(key, "1",
                    java.time.Duration.ofMinutes(5));
            if (Boolean.FALSE.equals(first)) {
                ((jakarta.servlet.http.HttpServletResponse) response)
                    .sendError(401, "重复请求");
                return;
            }
        } catch (NumberFormatException ignored) {}

        chain.doFilter(request, response);
    }
}
```

> 课程项目**可以不做**，但生产环境必做。**前端配合**：
> ```js
> headers: {
>   'X-Timestamp': Date.now(),
>   'X-Nonce': crypto.randomUUID(),
>   'Authorization': `Bearer ${token}`
> }
> ```

### 2.5 敏感字段脱敏

##### `SensitiveMaskUtil.java`
```java
package com.smarteldercare.common.security;

public final class SensitiveMaskUtil {
    private SensitiveMaskUtil() {}

    /** 13800001234 → 138****1234 */
    public static String maskPhone(String phone) {
        if (phone == null || phone.length() < 7) return phone;
        return phone.substring(0, 3) + "****" + phone.substring(phone.length() - 4);
    }

    /** 50000020000101001X → 500000********001X */
    public static String maskIdCard(String id) {
        if (id == null || id.length() < 10) return id;
        return id.substring(0, 6) + "********" + id.substring(id.length() - 4);
    }

    /** 张三 → 张* */
    public static String maskName(String name) {
        if (name == null || name.isEmpty()) return name;
        if (name.length() == 1) return name;
        return name.charAt(0) + "*".repeat(name.length() - 1);
    }
}
```

### 2.6 操作日志

**目标**：谁、什么时候、对哪条数据、做了什么——出事能追责。

##### 1) 注解 `@OperationLog`

```java
public @interface OperationLog {
```

##### 2) 切面 `OperationLogAspect.java`
```java
package com.smarteldercare.common.security;

import com.fasterxml.jackson.databind.ObjectMapper;
import jakarta.servlet.http.HttpServletRequest;
import lombok.extern.slf4j.Slf4j;
import org.aspectj.lang.ProceedingJoinPoint;
import org.aspectj.lang.annotation.Around;
import org.aspectj.lang.annotation.Aspect;
import org.aspectj.lang.reflect.MethodSignature;
import org.springframework.scheduling.annotation.Async;
import org.springframework.security.core.context.SecurityContextHolder;
import org.springframework.stereotype.Component;
import org.springframework.web.context.request.RequestContextHolder;
import org.springframework.web.context.request.ServletRequestAttributes;

import java.lang.reflect.Method;
import java.time.LocalDateTime;

@Slf4j
@Aspect
@Component
public class OperationLogAspect {

    private final ObjectMapper objectMapper = new ObjectMapper();

    @Around("@annotation(com.smarteldercare.common.security.OperationLog)")
    public Object around(ProceedingJoinPoint pjp) throws Throwable {
        long start = System.currentTimeMillis();
        Object result = pjp.proceed();
        long cost = System.currentTimeMillis() - start;

        try {
            saveLog(pjp, result, cost);
        } catch (Exception e) {
            log.warn("操作日志保存失败", e);
        }
        return result;
    }

    private void saveLog(ProceedingJoinPoint pjp, Object result, long cost) {
        MethodSignature sig = (MethodSignature) pjp.getSignature();
        Method m = sig.getMethod();
        OperationLog ann = m.getAnnotation(OperationLog.class);

        String username = "anonymous";
        try {
            var auth = SecurityContextHolder.getContext().getAuthentication();
            if (auth != null) username = auth.getName();
        } catch (Exception ignored) {}

        ServletRequestAttributes attrs =
                (ServletRequestAttributes) RequestContextHolder.getRequestAttributes();
        String ip = attrs != null ? attrs.getRequest().getRemoteAddr() : "";

        log.info("操作日志 user={} ip={} module={} action={} method={} uri={} cost={}ms",
                username, ip, ann.module(), ann.action(),
                sig.getDeclaringTypeName() + "." + m.getName(),
                attrs != null ? attrs.getRequest().getRequestURI() : "",
                cost);
        // TODO: 落库到 operation_log 表
    }
}
```

### 2.7 XSS 过滤

**目标**：防止老人姓名这种字段里被存 `<script>alert(1)</script>`。

##### `XssRequestWrapper.java`（用 Jakarta 包）
```java
package com.smarteldercare.common.security;

import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletRequestWrapper;

public class XssRequestWrapper extends HttpServletRequestWrapper {
    public XssRequestWrapper(HttpServletRequest request) {
        super(request);
    }
    @Override
    public String getParameter(String name) {
        return clean(super.getParameter(name));
    }
    @Override
    public String[] getParameterValues(String name) {
        String[] vals = super.getParameterValues(name);
        if (vals == null) return null;
        for (int i = 0; i < vals.length; i++) vals[i] = clean(vals[i]);
        return vals;
    }
    @Override
    public String getHeader(String name) {
        return clean(super.getHeader(name));
    }
    private String clean(String v) {
        if (v == null) return null;
        // 简单替换，生产用 Jsoup 或 OWASP Java HTML Sanitizer
        return v.replace("<", "&lt;")
                .replace(">", "&gt;")
                .replace("\"", "&quot;")
                .replace("'", "&#39;")
                .replace("&", "&amp;");
    }
}
```

|---|---|---|
| 浏览器里跨域读你的数据 | ✅ 能 | CORS |
| 黑客用 curl 直接打你接口 | ❌ 不能 | JWT 鉴权 |
| 偷到 token 后用别人账号 | ❌ 不能 | token 过期 + 黑名单 |
| 暴力破解密码 | ❌ 不能 | 限流 |
| SQL 注入 | ❌ 不能 | MyBatis #{} |
| 用户输入带 `<script>` | ❌ 不能 | XSS 过滤 |
| 数据库被脱 | ❌ 不能 | BCrypt |

**记住：CORS 是"用户体验项"，不是"安全项"。**
