# Smart ElderCare Data Platform Admin

智慧医养大数据公共服务平台云端后台管理系统。

本项目是一个面向医养服务机构的 Web 后台管理系统，主要用于管理老人档案、健康预警、评估报告、重点人群、医生账户、设备信息和数据看板。

## 项目定位

- 项目类型：Web 后台管理系统
- 使用场景：智慧医养、养老服务、健康数据管理
- 开发人数：4 人
- 项目负责人：前端页面、数据看板、接口对接、GitHub 仓库维护

## 推荐技术栈

- 后端：Spring Boot、MyBatis-Plus、MySQL、Redis
- 前端：Vue / React、Element Plus / Ant Design
- 图表：ECharts
- 接口测试：Apifox / Postman
- 协作工具：Git、GitHub

## 核心功能

- 登录与权限管理
- 老人档案管理
- 健康数据管理
- 健康预警管理
- 评估报告管理
- 重点人群管理
- 医生账户管理
- 设备管理
- 首页数据看板
- 报表统计
